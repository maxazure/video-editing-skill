# 14 导出剪映工程

> 把剪辑方案导出为剪映工程文件，在剪映中继续编辑和导出。

## 场景描述

视频剪辑方案已确定（render_config.json 已生成），但你不想直接用 ffmpeg 渲染，而是希望在剪映中继续编辑 —— 比如加特效、调色、精修转场、换字体等。

---

## 基础导出

```
帮我把剪辑方案导出成剪映工程文件，我想在剪映里继续编辑。
```

---

## 指定输出路径

```
把 render_config.json 导出成剪映工程，放到 ./jianying_draft 文件夹下。
```

---

## 导出后在剪映中打开

导出后的使用步骤：

1. 将导出的文件夹复制到剪映草稿目录：
   - **macOS**: `~/Movies/JianyingPro/User Data/Projects/com.lveditor.draft/`
   - **Windows**: `%APPDATA%/JianyingPro/User Data/Projects/com.lveditor.draft/`
2. 打开剪映，在草稿列表中即可看到该项目
3. 在剪映中可以继续：添加特效、调色、精修转场、换字体、导出视频

---

## 导出的内容

| 内容 | 说明 |
|------|------|
| 视频片段 | 所有选定的片段按顺序排列在主视频轨 |
| 片头封面 | 冻结首帧 + 标题文字叠加（支持自定义封面图） |
| 字幕 | 每句话作为独立字幕，单独轨道 |
| 片尾卡片 | 结尾文字卡片（如"感谢观看"） |
| BGM | 背景音乐轨道，自动设置音量 |
| 转场 | 片段间自动添加淡入淡出 |
| B-roll | 支持 B-roll 替换画面 |

---

## 注意事项

- 导出的工程使用**绝对路径**引用素材，不要移动原始视频文件
- 支持剪映专业版 6+ 版本
- 此功能与 ffmpeg 渲染**二选一**：需要快速出片用 render_final.py，需要精修用 export_capcut.py
