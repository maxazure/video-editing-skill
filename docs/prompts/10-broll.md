# 10 B-roll 画面替换

> 口播视频中用其他画面替换某些片段，保留原声。

## 场景描述

口播视频中提到某些东西（城市风景、产品、代码界面等），想用对应的 B-roll 画面替换掉你说话的镜头，但保留你的声音。

---

## 指定片段替换

```
我的口播视频有几个片段想换画面：
- 片段 3（讲到"深圳的夜景"）：换成 media/broll/shenzhen-night.mp4 的画面
- 片段 7（讲到"写代码"）：换成 media/broll/coding-screen.mp4 的画面
保留原来的声音，只替换画面。
```

---

## 指定 B-roll 起始位置

如果 B-roll 素材比较长，你想用其中某一段：

```
片段 5 的画面换成 media/broll/city.mp4，从第 10 秒开始截取。
```

---

## 配音 + B-roll 组合

当你只有录音没有口播画面时，可以配上 B-roll 做成视频：

```
我有一段纯配音录音 media/raw/voiceover.m4a，
还有一组 B-roll 素材在 media/broll/ 文件夹里。

帮我：
1. 识别配音内容
2. 根据每句话的内容匹配合适的 B-roll 画面
3. 生成最终视频，加上字幕
4. 竖屏格式
```

---

## 添加水印/叠加层

如果你想在全程叠加一个品牌 Logo 或水印：

```
帮我在视频全程叠加一个水印：media/assets/logo.png，放在右上角。
```
