# 12 字幕风格定制

> 6 种字幕风格 + 卡拉OK逐词高亮，匹配不同视频调性。

## 场景描述

不同类型的视频适合不同的字幕风格。活泼的用粗体大字，沉稳的用简约风，想炫酷的用卡拉OK逐词高亮。

---

## 查看可用风格

```
帮我看看有哪些字幕风格可以选？各是什么效果？
```

## 6 种字幕风格

| 风格 | 效果 | 适合 |
|------|------|------|
| `normal` | 白字 + 黑色描边 + 阴影 | 通用，最安全的选择 |
| `karaoke` | 逐词高亮（说到哪个词哪个词变黄） | 音乐、节奏感强的内容 |
| `bold_pop` | 粗体大字 + 重描边（MrBeast 风格） | 娱乐、搞笑、活泼 |
| `neon` | 黄色字 + 品红描边（赛博朋克风） | 科技、潮流 |
| `minimal` | 白字 + 无描边 + 淡阴影 | 文艺、简约、高级感 |
| `yellow_pop` | 黄色字 + 黑色描边 | 户外、嘈杂环境、高可见度 |

---

## 指定字幕风格

```
这个视频我想用 bold_pop 风格的字幕，像 MrBeast 那种粗体大字效果。
```

```
用 minimal 风格，简洁高级一点的。
```

```
用 neon 风格，科技感强一点。
```

---

## 卡拉OK逐词高亮

说到哪个词，那个词就高亮，类似卡拉OK效果：

```
字幕帮我用 karaoke 模式，逐词高亮。
```

> **注意**：卡拉OK模式需要逐词时间戳，AI 会自动在语音识别时启用 `--word-timestamps`。

## 指定字体

```
字幕字体用站酷快乐体（zcool-kuaile），看起来可爱一点。
```

```
字幕用霞鹜文楷（lxgw-wenkai），手写感的。
```

### 可用中文字体

| 字体 ID | 名称 | 风格 |
|---------|------|------|
| `noto-sans-sc` | 思源黑体 | 通用，默认 |
| `noto-serif-sc` | 思源宋体 | 正式、文化 |
| `lxgw-wenkai` | 霞鹜文楷 | 手写风 |
| `lxgw-wenkai-lite` | 霞鹜文楷 Lite | 手写风（轻量） |
| `zcool-kuaile` | 站酷快乐体 | 活泼可爱 |
| `zcool-qingke-huangyou` | 站酷庆科黄油体 | 潮流 |
| `zcool-xiaowei` | 站酷小薇体 | 优雅 |
