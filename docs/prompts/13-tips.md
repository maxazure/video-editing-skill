# 13 提示词技巧和常见问题

> 怎么写提示词效果最好，以及常见问题的解决方法。

---

## 提示词写作技巧

### 1. 说清楚文件位置

AI 需要知道你的文件在哪里。始终告诉它路径：

```
✅ 好：帮我处理 ~/Videos/2026-04-07/ 下面的视频
✅ 好：视频文件是 media/raw/talk.mp4
❌ 差：帮我处理那个视频（哪个？）
```

### 2. 说清楚语言

语音识别的语言会影响准确度：

```
✅ 好：中文口播，帮我做语音识别
✅ 好：识别语言用中文
❌ 差：帮我识别（什么语言？）
```

### 3. 说清楚输出要求

告诉 AI 你要的最终结果：

- **画面比例**：竖屏 / 横屏 / 正方形
- **目标平台**：抖音 / 小红书 / B站 / YouTube（AI 会自动选比例）
- **是否要字幕**：默认会加，不要的话记得说
- **字幕风格**：normal / karaoke / bold_pop / neon / minimal / yellow_pop
- **是否要封面**：需要的话说标题和副标题
- **是否要倍速版**：说明要哪些速度

### 4. 分步走还是一步到位

**新手建议分步**：

```
第一步：帮我做语音识别
（看完结果后）
第二步：片段 2、4、6 不错，帮我组合起来
（看完预览后）
第三步：加字幕和封面，输出竖屏
```

**老手可以一句话**：

```
帮我处理 media/raw/ 的口播视频，中文，去卡壳，加字幕，竖屏输出。
```

### 5. 追加和修改

处理完不满意？直接说哪里要改，不用重头来：

```
开头换成片段 7。
```

```
字幕风格换成 bold_pop。
```

```
再出一个 1.25 倍速版本。
```

## 常见问题

### Q: 语音识别不准怎么办？

```
帮我用 large-v3 模型重新识别，上次用的模型太小了准确度不够。
另外这些词注意一下不要识别错：Claude Code、OpenClaw、Remotion
```

### Q: 处理很慢怎么办？

```
先只处理前 3 分钟让我看看效果，确认没问题再处理全部。
```

或者指定用更小的模型加快速度：

```
语音识别用 small 模型就行，不需要太精确，速度优先。
```

### Q: 我只想要其中某几个片段？

```
我只要片段 2、5、8、11，其他的都不要。帮我把这四段拼起来。
```

### Q: 怎么加水印？

```
帮我加一个水印，图片是 media/assets/logo.png，放在右上角，全程显示。
```

### Q: 怎么加录制指示灯（REC 闪烁圆点）？

```
帮我在左上角加一个红色闪烁的 REC 圆点，模拟录像机的效果。
```

### Q: 视频有旋转问题（iPhone 竖拍）？

AI 会自动检测 iPhone 视频的 rotation 元数据并正确处理，一般不需要你操心。如果画面方向不对：

```
这个视频画面是倒的/是横的，帮我修正一下方向。
```

### Q: 转录文字有很多错别字？

```
帮我重新检查一遍 transcript，把识别错误的词都修正一下。
特别注意这些容易错的：
- "小红树" 应该是 "小红书"
- "检映" 应该是 "剪映"
- "cloud ops" 应该是 "Claude Opus"
```

### Q: 怎么添加章节进度条？

```
帮我给视频加上章节进度条，章节按内容自动划分，
在画面底部显示当前章节名和进度。
```
