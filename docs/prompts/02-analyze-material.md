# 02 分析素材并制定剪辑方案

> 让 AI 分析你的多条素材，判断哪些能用，怎么组合最好。

## 场景描述

你拍了好几条视频，不确定哪些能用、哪些需要重录、应该怎么组合。让 AI 帮你全面分析，给出剪辑建议。

---

## 全面分析

```
我在 media/raw/ 下面有 5 个视频文件，是今天拍的关于"如何学编程"这个主题的口播。
帮我全部做语音识别，然后分析一下：
1. 每个视频分别讲了什么内容
2. 哪些片段讲得好（有吸引力、信息量大、没有卡壳）
3. 哪些片段有问题（卡壳、口误、重复、内容空洞）
4. 建议怎么组合这些片段，能剪出几条短视频
5. 每条短视频的建议标题
```

---

## 按主题拆分方案

```
transcript 文件已经有了，帮我重新分析一下选片方案。
这次我想把内容分成 3 条短视频，每条控制在 1 分钟左右：
- 第一条做开头的 hook，吸引关注
- 第二条讲核心干货
- 第三条做总结，加上呼吁关注
```

---

## 对比多次拍摄

有时候同一段内容你拍了好几遍，想让 AI 帮你选最好的那一遍：

```
media/raw/ 里面的 take1.mp4、take2.mp4、take3.mp4 是同一段内容拍了三遍。
帮我都识别一下，对比分析哪个版本最好：
- 哪个口误最少
- 哪个语气最自然
- 哪个节奏最好
帮我选一个最好的版本来用。
```

---

## 跨视频混合选片

```
这 3 个视频虽然是分开拍的，但讲的是同一个话题。
帮我从所有视频里跨视频选片，找出每个知识点讲得最好的那一遍，
组合成一个最终版本。
```

---

## 查看素材库状态

如果你已经有一个建好的素材库，想看看里面有什么：

```
帮我看看素材库里都有哪些视频，列一下每个文件的时长和内容概要。
```
