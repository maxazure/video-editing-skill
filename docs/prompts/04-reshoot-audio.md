# 04 补录音频

> 画面没问题，只替换声音。适合口误修复、配音替换等场景。

## 场景描述

视频画面拍得不错，但有些地方说错了话，或者背景噪音太大。你只想重新录一段音频替换上去，画面保持不变。

---

## 让 AI 分析音频问题

```
帮我分析一下这个视频的语音质量。我想知道：
1. 哪些地方说得磕磕巴巴需要重新配音
2. 哪些地方有明显的口误（说错话了）
3. 哪些地方背景噪音太大

给我列一个补录音频的清单，每条写上：
- 对应的片段编号和时间范围
- 原来说了什么（识别结果）
- 建议重新录什么内容
- 大概需要多少秒
```

### AI 输出示例：

> **音频补录 #1：片段 5（01:23 - 01:35）**
> - 原始内容："那个...就是说...这个方法呢...就是很...很有效的"
> - 问题：填充词过多，听感很差
> - 建议重录："这个方法非常有效，我自己用了三个月就看到明显效果了。"
> - 时长：约 6 秒
>
> **音频补录 #2：片段 8（02:15 - 02:28）**
> - 原始内容："React 可以用来做...呃...后端开发"
> - 问题：口误（React 是前端框架）
> - 建议重录："React 可以用来做前端开发，搭配 Node.js 还能做全栈。"
> - 时长：约 5 秒

---

## 用新音频替换

补录完音频后，告诉 AI 怎么替换：

```
我重新录了几段音频，放在 media/raw/audio-reshoot/ 里。
帮我用这些新录音替换原视频对应片段的声音，画面保持不变：
- reshoot_01.m4a 替换片段 5 的音频
- reshoot_02.m4a 替换片段 8 的音频
替换后重新渲染最终视频。
```

---

## 单独替换某一句

如果只是某一句说错了：

```
片段 8 里我把"前端"说成了"后端"，我重新录了一段正确的音频 fix_seg8.m4a。
帮我只替换这个片段的声音，画面不动，重新输出视频。
```

---

## 纯配音替换（全部音频替换）

有时候你想完全用新录音替换原视频的声音（比如原来是现场录音，声音质量不好，想在安静的房间重新录一遍）：

```
我在安静的环境里重新录了一遍完整的配音，文件是 media/raw/voiceover-clean.m4a。
帮我用这段新录音完全替换原视频的声音，画面保持原来的。
需要重新做语音识别来对齐时间轴。
```
