# 11 批量处理

> 多条素材走同样的流程，一次处理完。

## 场景描述

你一次拍了好几条视频，每条都要做语音识别、去卡壳、加字幕、输出短视频。逐条说太麻烦，一次搞定。

---

## 批量处理全流程

```
media/raw/ 下面有 8 个视频文件，都是今天拍的口播。
帮我批量处理：
1. 全部提取音频
2. 全部做语音识别（中文）
3. 每个视频单独分析，去掉卡壳和口误
4. 每个视频单独输出一条短视频
5. 都用竖屏格式，加字幕，生成封面
6. 输出到 media/output/
```

---

## 只做识别，不做剪辑

有时候你只想先把所有素材识别一遍，之后再决定怎么剪：

```
帮我把 media/raw/ 下面所有视频都做语音识别（中文），
生成 transcript 文件就行，先不用剪辑。
```

---

## 统一风格批量输出

```
media/raw/ 下面 5 个视频，帮我都处理成短视频，统一用以下设置：
- 字幕风格：bold_pop
- 封面风格：news
- 背景音乐：media/bgm/upbeat.mp3，音量 12%
- 竖屏格式
- 每条都出 1x 和 1.25x 两个速度
```
