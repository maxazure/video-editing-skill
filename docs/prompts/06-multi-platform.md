# 06 多平台导出

> 一条视频，一键导出抖音、Instagram、YouTube 等多种比例和速度版本。

## 场景描述

视频剪好了，但你要发多个平台，每个平台的画面比例不一样。同时可能还需要不同倍速版本。

---

## 三平台全导出

```
视频已经剪好了，帮我导出三个版本：
1. 竖屏 9:16 —— 发抖音和小红书
2. 正方形 1:1 —— 发 Instagram
3. 横屏 16:9 —— 发 YouTube 和 B 站
```

---

## 多比例 + 多倍速

```
帮我导出以下版本：
- 竖屏 9:16：1x 和 1.25x
- 横屏 16:9：1x
一共 3 个文件。
```

---

## 只要竖屏多倍速

```
帮我把剪好的视频导出竖屏格式，同时出 1x、1.25x、1.5x 三个速度版本。
```

---

## 指定某个平台

直接说平台名，AI 会自动选对应比例：

```
帮我出一个发抖音的版本和一个发 B 站的版本。
```

| 平台 | 比例 | 分辨率 |
|------|------|--------|
| 抖音 / TikTok / 小红书 / YouTube Shorts | 9:16 竖屏 | 1080x1920 |
| Instagram | 1:1 正方形 | 1080x1080 |
| YouTube / B站 | 16:9 横屏 | 1920x1080 |
