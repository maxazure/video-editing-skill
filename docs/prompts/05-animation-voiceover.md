# 05 动画配音视频（Remotion）

> 把录音变成动画解说视频。效果类似动态 PPT，有文字动效、渐变背景、进度条。
> 适合教程讲解、知识科普、脱口秀等只有音频没有画面的场景。

## 场景描述

你录了一段音频（配音、解说、教程讲解、脱口秀等），想配上动画效果做成视频发布。不需要真人出镜，用动态文字 + 视觉效果就能把内容表达清楚。

---

## 基础用法：录音变视频

```
我录了一段音频解说，文件是 media/raw/explain-python.m4a，
内容是讲解 Python 基础知识的，大概 3 分钟。
帮我用 Remotion 做成一个动画视频：
- 先做语音识别
- 然后生成动态文字视频配合我的录音
- 风格活泼一点
- 竖屏格式，发抖音用
```

---

## 教程类：模拟 PPT 讲解效果

这是最常用的场景——你录了一段教程讲解，想做成类似 PPT 演示但更生动的视频：

```
我录了一段教程音频 media/raw/tutorial-react.m4a，大概 5 分钟，
讲的是 React Hooks 的用法。

帮我做成动画解说视频，效果要像动态 PPT：
- 每讲到一个知识点，在画面上显示对应的要点文字
- 重要概念用醒目的动画效果（比如 bounce 或 slam）
- 普通讲述用平缓的淡入效果
- 按内容自动分章节
- 每个章节用不同的背景颜色区分
- 底部显示章节进度条

竖屏和横屏各出一个版本。
```

## 知识科普类

```
我有一段科普音频 media/raw/science-explain.wav，
讲的是"为什么天空是蓝色的"，2 分钟左右。

做成动画视频，要求：
- 风格用 calm（沉稳科普风）
- 关键术语（瑞利散射、波长等）用大字体特殊强调
- 背景用深色渐变，显得专业
- 加底部进度条
- 竖屏格式
```

---

## 脱口秀 / 段子类

```
我有一段脱口秀录音 media/raw/standup.wav。
帮我生成脱口秀风格的文字动画视频：
- 自动识别笑点，笑点用 shake 或 bounce 动画强调
- 风格用 energetic
- 背景颜色自动循环变化
- 节奏跟上我说话的速度
- 加上音频波形动画
```

## 指定动画风格和细节

如果你想精细控制动画效果：

```
帮我用 Remotion 生成配音动画视频。
音频文件：media/raw/tech-talk.wav

具体要求：
- 风格：calm（沉稳）
- 关键句子用大字体突出显示
- 背景：渐变色，自动切换
- 加底部进度条
- 加音频波形动画
- 场景切换时要有转场效果（淡入淡出）
- 字体用思源黑体
```

### 可选的风格预设

| 风格 | 效果 | 适合 |
|------|------|------|
| `default` | 平衡 | 通用 |
| `calm` | 沉稳、柔和 | 教程、科普、文化 |
| `energetic` | 活泼、动感 | 脱口秀、段子、娱乐 |

### 可用的文字动画效果

| 动画 | 效果 | 适合场景 |
|------|------|----------|
| `fadeIn` | 淡入 | 普通叙述 |
| `springIn` | 弹性出现 | 轻快的要点 |
| `scaleUp` | 放大出现 | 重要概念 |
| `bounce` | 弹跳 | 强调/笑点 |
| `shake` | 抖动 | 惊讶/笑点 |
| `slam` | 砸入 | 关键观点 |
| `wave` | 波浪 | 柔和过渡 |
| `glitch` | 故障风 | 科技感 |
| `typewriter` | 打字机 | 代码/引用 |

## 批量生成

如果你有多段音频都要做成视频：

```
media/raw/episodes/ 下面有 5 个音频文件，每个都是一期内容。
帮我全部做成动画视频：
- 风格统一用 calm
- 竖屏格式
- 每个视频自动分章节
- 输出到 media/output/episodes/
```

---

## 注意事项

1. **需要 Node.js**：Remotion 动画视频需要安装 Node.js 和 npm
2. **首次使用较慢**：第一次会安装 Remotion 依赖，之后就快了
3. **竖屏优先**：动画视频天然适合竖屏，文字更大更清晰
4. **音频质量**：录音尽量在安静环境，音频质量越好，识别越准，最终效果越好
